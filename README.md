# RealtimeNotify
RealtimeNotify - это проект, обеспечивающий передачу реального времени уведомлений пользователям с использованием технологии WebSocket. Этот проект позволяет отправлять уведомления в режиме реального времени, обеспечивая мгновенное взаимодействие между сервером и клиентом.
## Возможности 
- Мгновенная доставка уведомлений: RealtimeNotify использует WebSocket для доставки уведомлений от сервера к клиенту мгновенно, без необходимости перезагрузки страницы.
- Группировка уведомлений: Пользователи могут быть добавлены в группы, и уведомления могут быть отправлены определенным группам пользователей.
- Гибкая интеграция: RealtimeNotify может быть легко интегрирован в существующие проекты, позволяя добавить мгновенные уведомления без значительных изменений кода.
## Технологии
- Django: Фреймворк на языке Python, используемый для разработки серверной части приложения.
- Django Channels: Расширение Django, которое позволяет использовать WebSocket для обработки подключений в режиме реального времени.
- JavaScript: Используется для обработки подключений WebSocket на стороне клиента.
- Celery: Распределенный фреймворк для обработки задач, позволяющий отправлять уведомления асинхронно.
- Redis: Хранилище данных в оперативной памяти, используемое в качестве посредника сообщений для Celery и для канального слоя в Django Channels.
## Установка и настройка
1. Склонируйте репозиторий RealtimeNotify с GitHub.
2. Установите зависимости, выполнив команду `pip install -r requirements.txt`.
3. Создайте и примените миграции базы данных, используя команды python manage.py makemigrations и python manage.py migrate.
4. Запустите Celery `celery -A realtimenotify worker --loglevel=info -P eventlet`
5. Запустите сервер разработки Django, выполнив команду python manage.py runserver. Откройте веб-браузер и перейдите по адресу `http://localhost:8000`, чтобы получить доступ к приложению RealtimeNotify.
6. Подключитесь как `python manage.py shell` и создайте экземпляр модели `Notification` из `app/models.py` и далее импортируя функию `send_notification` из `app/tasks.py` можете отправить уведомление под заданным id
